# ALAKIR Gallery — v6

גלריית אמנות בעברית עם ממשק רכישה מלא וממשק ניהול אינטראקטיבי.
מחובר ל-**iCount** לסליקה וחשבוניות מס אוטומטיות.

---

## מה יש בחבילה

```
alakir_v6/
├── index.html              ← עמוד הבית
├── gallery.html            ← הגלריה עם רכישה (54 יצירות)
├── admin.html              ← ממשק ניהול
├── 404.html                ← דף שגיאה
├── data/
│   └── gallery-data.json   ← מקור האמת — היצירות, מחירים והגדרות
├── assets/                 ← תמונות, פונטים, רקעים
├── server/                 ← שרת Node אופציונלי (לסנכרון בין מכשירים)
│   ├── server.js
│   ├── package.json
│   └── .env.example
├── README.md               ← הקובץ הזה
└── DEPLOYMENT.md           ← מדריך התקנה ופרסום מפורט
```

---

## איך זה עובד

הקונה לוחץ "הוסף לסל" → ממלא שם, מייל, טלפון, כתובת → לוחץ "המשך לתשלום"
→ מועבר לעמוד תשלום מאובטח של **iCount** → מקליד פרטי כרטיס אשראי
→ iCount מנפיק חשבונית מס אוטומטית → הקונה חוזר לאתר עם הודעת תודה
→ הכסף יושב בחשבון iCount, מועבר אוטומטית לבנק תוך 1-2 ימי עסקים

**אין צורך בשרת לסליקה.** הקוד שלנו רק בונה את ה-URL לעמוד התשלום של iCount,
מעביר אליו, ומחזיר את הקונה אחרי תשלום מוצלח.

## איך מתחילים

### 1️⃣ בדיקה מקומית
פתחו את `admin.html` — סיסמה: `alakir2026` — שנו מחירים, סטטוסים וכותרות,
ראו שהכל מתעדכן ב-`gallery.html`.

### 2️⃣ פתיחת חשבון iCount
היכנסו ל-<https://www.icount.co.il> → 45 יום ניסיון חינם → צרו עמוד סליקה
→ העתיקו את הקוד שלו (החלק שאחרי `icount.co.il/m/`).

### 3️⃣ הזנת הקוד באתר
ב-`admin.html → הגדרות → תשלום וסליקה` הדביקו את הקוד.
הסליקה כעת אמיתית.

### 4️⃣ העלאה לאוויר
מעלים את כל התיקייה ל-Cloudflare Pages / Netlify / VPS — מה שמתאים לכם.
ראו `DEPLOYMENT.md` למדריך מפורט.

## מה אפשר לעשות באדמין

- **לוח בקרה:** סטטיסטיקות, יצירות אחרונות, הזמנות אחרונות
- **יצירות:** עריכה, מחיקה, הוספה, החלפת תמונות, חיפוש, סינון, פעולות בכמות
- **הזמנות:** טבלה מלאה, פרטי לקוח, ייצוא CSV
- **הגדרות:** כותרות, צבעים, iCount, סיסמה, ייצוא/ייבוא נתונים

**סיסמת ברירת מחדל:** `alakir2026` (שנו אותה בהגדרות → אבטחה!)

## מצבי הפעלה

| מצב | מתי לבחור | מה צריך |
|-----|----------|---------|
| **מקומי** | בדיקה אישית, הכנה לפני שינוי תוכן | רק פתיחת `admin.html` בדפדפן |
