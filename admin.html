מדריך מהיר · Quick guide

איך זה עובד

כל שינוי כאן — מחיר, סטטוס, תמונה — נשמר אוטומטית ומופיע מיד בעמוד הגלריה. אין צורך לשמור ידנית מלבד בעמוד ההגדרות.

קיצורי מקלדת

Esc — סגירת חלון
Cmd/Ctrl + S — שמירה
/ — מיקוד בחיפוש

מצבי הפעלה

Local — הנתונים נשמרים בדפדפן בלבד. מצוין לעבודה אישית.

Server — שרת Node מאחורי הקלעים שומר את הנתונים בקובץ ומסונכרן בין מכשירים. ראה DEPLOYMENT.md.

תמיכה בתשלומים

האתר מחובר ל-iCount — סליקה ישראלית עם חשבונית מס אוטומטית.
תומך בויזה, מאסטרקרד, אמריקן אקספרס, ישראכרט, Apple Pay ו-Google Pay.
ראה הוראות הפעלה ב-DEPLOYMENT.md.
